Ques. Difference between flex row and column.

Flex Row and column are the properties of CSS related to Flexbox having Flex Container and Flex Items . Flex Row is a value of flex direction property which is used to put the Flex Items in row form while column value is used to align the items in the form of column.

Ques. Difference between absolute and relative positioning

In CSS absolute and relative position are values of position property , in absolute postion the element whose position is absolute the element is postioned relative to its parent element having postion value relative , absolute etc , except Static while relative positioning positions the element relative to its normal postion. All these property will be efected accordingly using top , left , right , bottom properties.

Ques. What is grid-template-columns used for?

In CSS grid-template-columns is a property of CSS Grids , its is used to specify the number of columns and also used to specify the width of the columns in the Grid.

Ques. What is the z-index in CSS?

In CSS Z-index property is used to determine the stack order of elements , Z-index usually works for positioned elememnts when there are more than one element on same postion then which element will be shown first of web page . The Z-index value can be negative value also . The Greater the value is the higher priority will be given to element in that stack order .

Ques. What is the difference between padding and margin?

Margin and Padding are two property in CSS which are part box-model, Padding property defines the space between the content and border of the element. Padding value should not be negative value , while Margin is used create space around elements, outside of any defined borders. Margins values can be negative values.
